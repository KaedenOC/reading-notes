# Practice In The Terminal

**The Command Line** - What is it, how does it work and how do I get to one?

- Text based interface to the system, and you can enter commands and get feedback to you in text.

- prompts, commands, and command line arguments. You must put spaces between the command and the first command line argument.

- *Mac Users* - 'command + space', then type Terminal and it will show up. *Linux* - right click on the desktop and 'Open in terminal'. *Windows* - will need an SSH (Secure Shell) client.

**Basic Navigation** - An introduction to the Linux directory system and how to get around it.

- *pwd* - print working directory - this will tell us our current working directory or current position.

- *ls* - list - this will show us what is there

- ~ - tilde - shortcut for the home directory (to go back home).

- . - dot - reference to current directory.

- .. - dotdot - reference to the parent directory.

- *cd* - change directory - use this command to move around the system.

**More About Files** - Find out some interesting characteristics of files and directories in a Linux environment.

- Everything is a file.

- *file.exe* - an executable file, or program.

- *file.txt* - a plain text file.

- *file.png*, *file.gif*, *file.jpg* - an image.

- **Hidden Files and Directories** - to make a file or directory hidden use a '.' (dot) in front of the file name. *ls* will not show hidden files or directories, we would want to use *-a*. ex: ls -a

**Manual Pages** - Learn how to make the most of the Linux commands you are learning.

- set of pages that explain every command available on your system.

- invoke the manual pages using 'man < command to look up>'.

- keyword search for all manual pages containing the given search term 'man -k < search term>'.

- Within a manual page, perform a search for 'term' using '/< term>'.

- after a search use 'n' to select the next found item.

**File Manipulation** - How to make, remove, rename, copy and move files and directories.

- *mkdir* - make directory - use to create a directory.

- *rmdir* - remove directory - use to delete a directory.

- *touch* - create a file

- *cp* - copy - use to copy a file or directory.

- *mv* - move - use to move a file or directory, which can also be used to rename.

- *rm* - remove - use to delete a file.
